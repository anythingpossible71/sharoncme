# Upload Team Images to CrunchyCone Storage

## Instructions

To upload team member images to CrunchyCone storage:

1. Go to `/admin/media` in your admin panel
2. Upload the following images to the `team` folder:

   - **alex-johnson.jpg** - Alex Johnson (Founder & CEO)
   - **sarah-chen.jpg** - Sarah Chen (Lead Developer)  
   - **marcus-rodriguez.jpg** - Marcus Rodriguez (Design Lead)
   - **emily-watson.jpg** - Emily Watson (Product Manager)

3. Make sure to:
   - Set visibility to **Public**
   - Upload to the **team** folder
   - Use the exact filenames listed above

## Image Sources

You can download placeholder images from:
- Unsplash: https://unsplash.com/s/photos/portrait
- Or use any professional headshots

The component expects images at:
- `/api/storage/files/team/alex-johnson.jpg`
- `/api/storage/files/team/sarah-chen.jpg`
- `/api/storage/files/team/marcus-rodriguez.jpg`
- `/api/storage/files/team/emily-watson.jpg`
